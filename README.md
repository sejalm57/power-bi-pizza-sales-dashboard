# Power-BI-Pizza-Sales-Dashboard
🍕 Power BI – Pizza Sales Dashboard
This project presents an interactive Pizza Sales Dashboard developed using Power BI, aimed at analyzing sales performance, customer ordering patterns, and product-level insights from a business perspective.
The dashboard helps stakeholders quickly understand revenue trends, order behavior, and product performance through well-designed KPIs and visuals.

🎯 Project Objective
The objective of this project is to analyze pizza sales data to uncover meaningful business insights and support data-driven decision-making.
This dashboard focuses on:
Sales performance analysis
Customer order behavior
Product and category performance
Trend analysis over time

❓ Problem Statement
Before building this dashboard, it was difficult to analyze pizza sales data effectively due to the lack of a consolidated and visual reporting system.
The business needed clarity on:
Which pizzas generate the highest and lowest revenue
Peak order days and seasonal demand patterns
Sales contribution by pizza category and size

Overall business performance using KPIs

📊 Key Features of the Dashboard
🔹 KPIs

Total Revenue
Total Orders
Total Pizzas Sold
Average Order Value
Average Pizzas per Order

🔹 Sales & Trend Analysis

Daily order trends by day of the week
Monthly order trends to identify seasonality
Busiest days and peak order times

🔹 Product Performance

Sales contribution by pizza category
Sales contribution by pizza size

Top 5 and Bottom 5 pizzas by:
Revenue
Quantity
Total Orders

🔹 Interactivity

Pizza Category slicer
Date range slicer

Navigation buttons (Home / Best–Worst) for smooth page navigation

🛠 Tools & Technologies Used

Interactive Visuals & Slicers
